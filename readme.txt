Plugin de integración de factura electrónica miPOS en WooCommerce.

== Description ==
Este plugin permite integrar la facturación electrónica en tu tienda WooCommerce, conectándose directamente al Ministerio de Hacienda de Costa Rica y al sistema miPOS. Genera facturas electrónicas automáticamente para cada compra en tu tienda online, simplificando el cumplimiento tributario en Costa Rica. **Para usar este plugin debe adquirir el servicio en [https://www.jarscr.com/facturaelectronica](https://www.jarscr.com/facturaelectronica) y tener una cuenta en [https://mipos.co.cr](https://mipos.co.cr).**

== Installation ==

1.  Adquiere el servicio en [https://www.jarscr.com/facturaelectronica](https://www.jarscr.com/facturaelectronica).
2.  Crea una cuenta en [https://mipos.co.cr](https://mipos.co.cr).
3.  Descarga el plugin desde el repositorio de WordPress.
4.  Sube el plugin a tu directorio /wp-content/plugins/ o instálalo directamente desde la página de plugins de WordPress.
5.  Activa el plugin a través del menú "Plugins" en WordPress.
6.  Ve al panel de configuración del plugin en WooCommerce y ingresa tus datos de acceso.

== Frequently Asked Questions ==

= ¿Cómo funciona el plugin? =
El plugin se conecta a la API del servicio que adquirió en [https://www.jarscr.com/facturaelectronica](https://www.jarscr.com/facturaelectronica) el cual se conecta al Ministerio de Hacienda de Costa Rica y al sistema miPOS ([https://mipos.co.cr](https://mipos.co.cr)). Para configurar el plugin debe tener una cuenta en miPOS.

= ¿Qué datos necesito para usar el plugin? =
Necesitas los datos de acceso, los cuales puedes adquirir fácilmente en nuestro [sitio web](https://www.jarscr.com/facturaelectronica). **También debe tener una cuenta en [https://mipos.co.cr](https://mipos.co.cr).**

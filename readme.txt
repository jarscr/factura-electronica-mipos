Plugin de integración de factura electrónica miPOS en WooCommerce.

== Description ==
Este plugin permite integrar la facturación electrónica en tu tienda WooCommerce, conectándose directamente al Ministerio de Hacienda de Costa Rica y al sistema miPOS. Genera facturas electrónicas automáticamente para cada compra en tu tienda online, simplificando el cumplimiento tributario en Costa Rica.

== Frequently Asked Questions ==

= ¿Cómo funciona el plugin? =
El plugin se conecta a una API que genera tu factura electrónica basada en los datos de cada compra en tu tienda online.

= ¿Qué datos necesito para usar el plugin? =
Necesitas los datos de acceso, los cuales puedes adquirir fácilmente en nuestro [sitio web](https://www.jarscr.com).
